Instructions

  1. Prepare Characters: Existing characters are available from All Characters. If none are suitable, use Add Character to add the new ones you need. Characters have a name, description, personalityy and motivation and can have an associated picture
  2. Add Story: Use the Add Story option. Each story needs a header comprising an author, a title, a note and system instructions. The note is just for reference and displayed in the the All Stories option. System instructions guide the AI and can be changed
  3. Build the Story: If you add a post, the AI will respond. You can add characters to posts to give the AI a good background. You can edit the last response from the AI and previous prompts. Editing a prior prompt will delete subsequent prompts and AI responses
  4. Maintain Story: Use All Stories to select a story and modify it. You can also use this form to delete a story
